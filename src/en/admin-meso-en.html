MESO admin panel for grocery stores

MESO — is a unified showcase for convinience stores nearby that want to deliver their goods. I’ve designed an app and an admin panel for them.

It’s a second version of the admin panel — now with growth area (the first one was made at bench scale for hypothesis testing). This version allows to track couriers, edit catalog and observe customer tastes.

The Form

Thanks to the first version we understood some of the habits of users better. One of them is — many users manage shops on laptops with old screens, so large tables were initially adapted for low resolutions.
